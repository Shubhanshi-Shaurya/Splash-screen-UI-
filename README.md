## Splash Screen UI 

A simple and elegant **Splash Screen** built using HTML, CSS, and JavaScript. This project displays a loading or welcome screen before transitioning to the main content or application.

---

## Features
- Clean and modern splash screen design
- Automatic transition after a delay
- Smooth animations (fade/scale/loader)
- Lightweight and fast
- Responsive layout

---

## Technologies Used
- **HTML** 
- **CSS**

## Future Improvements
- Add logo animation (SVG or GIF)

## How It Works
- When the page loads, the splash screen is displayed.
- A timer or animation runs for a few seconds.
- After completion, the splash screen:
- Fades out / disappears
- Redirects or reveals the next content (optional)

---

## Use Cases
- App or website loading screens
- Portfolio intro screens
- Branding or logo animation display
- Preloaders for web application
